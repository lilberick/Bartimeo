# Reconocer personas
## Instalación
```
$ pip3 install opencv-python 
$ pip3 install face-recognition
$ pip3 install picamera 
```
## Ejecución
1. Ejecutamos
	```
	$ python codigo.py
	```
2. Enfocamos la cámara en la cara de la persona o foto
